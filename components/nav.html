# 实现逻辑

  • 组件通信,在父组件使用 provide
provide() {
    return {
      root: this,
      vertical: this.vertical
    };
  },

在子组件或孙组件使用 inject 注入

inject: ["root", "vertical"],
  • 没有什么类
&:not(.vertical) {
  &::after {
    content: "";
    position: absolute;
    bottom: 0;
    left: 0;
    border-bottom: 1px solid $blue;
    width: 100%;
  }
}
